AviJet

AviJet is a fully functional full-stack Aviator-style gaming platform featuring real-time gameplay, secure authentication, wallet management, payment processing, and automated win/loss calculations.

🌐 Live Demo: https://avijet.netlify.app

Features

- Real-time Aviator gameplay
- User registration and login system
- Secure authentication and account management
- Email-based password recovery
- Wallet balance management
- Payment gateway integration for deposits
- Automatic balance synchronization
- Real-time win/loss calculations
- Transaction tracking and management
- Responsive design for desktop and mobile devices
- Fully connected frontend and backend architecture

Backend Functionality

The backend is responsible for:

- User authentication and authorization
- Payment processing and verification
- Wallet balance updates
- Game result calculations
- Win/loss settlements
- Database management
- Email services and password reset functionality
- Secure API handling
